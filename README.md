# Sistema de Reservas de Hotel

Sistema de reservas de hotel desenvolvido em Python com MongoDB com controle de hóspedes, reservas e quartos, relatórios de ocupação, número de reservas por mês e relatório de listagem de hóspedes. 

Integrantes do grupo: Davi Pereira de Sousa, Gabriela Gave Gavi, José Luiz dos Santos Azeredo, Pedro Henrique Bispo, Pedro Henrique Ferreira Bonela.
